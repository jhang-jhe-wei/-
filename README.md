# Star War
## Introduction  
  This is a Star Wars game made with java
## Video
  [![Star War](https://i.imgur.com/6GrmkrH.png)](https://youtu.be/bRKvdPA267w)
## Requirements
  Java SE ~>1.8.1
## How to play it?
  Keyborad  | Function
  :--------:|:--------:
  Upwards   |   UP
  Downwards |   Down
  Leftwards |   Left
  Rightwards|   Right
  Z         | scattering
  X         | Burst
  Shift     | safety mask
